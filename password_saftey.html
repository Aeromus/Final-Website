Password Saftey

Entropy

Password entropy is a measurement of how unpredictable a password is. Password entropy is based on the character set used (which is expansible by using lowercase, uppercase, numbers as well as symbols) as well as password length. This comic below by XKCD does a good job of explaining password entropy and also offers a way to make passwords easier to remember

Password Reuse

Another major security risk when it comes passwords is the reuse of passwords across multiple devices / accounts. The risk with this is that if a password is cracked / discoverd it can easily be tried (and any permutations) on many seperate accounts / devices. Again a good comic from Randall Munroe at XKCD explains this here.
